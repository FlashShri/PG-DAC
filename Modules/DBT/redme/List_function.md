# List functions
    * coalesce()
    * concat()
    * greatest()
    * least()
```SQL
mysql> SELECT greatest( 23, 45,2,45,666, 66 ) ;
+---------------------------------+
| greatest( 23, 45,2,45,666, 66 ) |
+---------------------------------+
|                             666 |
+---------------------------------+
1 row in set (0.00 sec)

mysql> SELECT least ( 23, 45,2,45,666, 66 ) ;
+-------------------------------+
| least ( 23, 45,2,45,666, 66 ) |
+-------------------------------+
|                             2 |
+-------------------------------+

mysql> SELECT least ( 23, 45,2,45,666, 66 , -45);
+------------------------------------+
| least ( 23, 45,2,45,666, 66 , -45) |
+------------------------------------+
|                                -45 |
+------------------------------------+
```
